###### Customizing Theme Colors ğŸ¨

Warriorfolio 2 uses Tailwind CSS via Vite. Modify theme colors in `tailwind.config.js`:

```javascript
theme: {
    extend: {
        colors: {
            primary: colors.purple,
            secondary: colors.zinc,
            tertiary: colors.rose,
        }
    },
},
```

Default color scheme:
- `primary`: Purple
- `secondary`: Zinc
- `tertiary`: Rose
